DICOM Conformance Statements

These DICOM conformance Statements provide details about the DICOM Service Classes, Information Objects and Communication protocols supported by Toshiba's products. This information is intended to facilitate checking compatibility and integrating these products within a DICOM compliant hospital network. Note that some variability is permitted within the DICOM Standard, so connectivity confirmation is still advised.
